# LobeHub v2.0 — Group Chat & Multi-Agent Collaboration 🎉

January marks the landmark release of LobeHub v2.0, introducing powerful multi-agent group chat capabilities, refined model settings, and a streamlined authentication experience.

## 🌟 Key Updates

- 🎉 LobeHub v2.0: A major version upgrade with redesigned architecture and enhanced features
- 👥 Group chat: Multi-agent collaboration with group topics, shared avatars, and message compression controls
- 🔐 SSO-only mode: New `AUTH_DISABLE_EMAIL_PASSWORD` option to enforce SSO-only authentication
- 🤖 Model updates: Updated Kimi K2.5 and Qwen3 Max Thinking models, plus Gemini 2.5 streaming fixes
- 🛠️ Enhanced model settings: New ExtendParamsTypeSchema for more flexible model configuration
- 🖥️ Desktop improvements: Fixed download URLs, improved onboarding flow, and updated Docker image for v2

## 💫 Experience Improvements

Improved cron job UI, fixed usage table display, optimized artifact prompts, resolved mobile inbox agent issues, improved speed metrics with tool call tracking, and fixed Klavis onboarding connection timeout.
