# LobeHub v2.0 — 群聊与多智能体协作 🎉

一月，LobeHub 迎来了里程碑式的 v2.0 版本发布，引入了强大的多智能体群聊功能、优化的模型设置以及更流畅的认证体验。

## 🌟 重要更新

- 🎉 LobeHub v2.0：全新架构设计与功能增强的大版本升级
- 👥 群聊功能：支持多智能体协作，包括群组话题、共享头像和消息压缩控制
- 🔐 SSO 专用模式：新增 `AUTH_DISABLE_EMAIL_PASSWORD` 选项，支持强制 SSO 登录
- 🤖 模型更新：更新 Kimi K2.5 和 Qwen3 Max Thinking 模型，修复 Gemini 2.5 流式传输
- 🛠️ 增强模型设置：新增 ExtendParamsTypeSchema，提供更灵活的模型配置
- 🖥️ 桌面端改进：修复下载链接、优化引导流程，更新 Docker 镜像适配 v2

## 💫 体验优化

改进定时任务 UI、修复用量表格显示、优化 Artifact 提示词、解决移动端收件箱问题、改进速度指标中的工具调用追踪，并修复 Klavis 引导连接超时问题。
